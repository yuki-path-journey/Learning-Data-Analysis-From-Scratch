# Learning Data Analysis From Scratch 🌲🏡

## 🕵️‍♂️ このリポジトリについて
Kaggleの「Ames Housing（住宅価格予測）」を題材に、データ分析を基礎から学ぶための復習記録です。
「分析のアイディアは浮かぶけれど、コードが難しい！」という壁を突破するため、一つ一つのステップを日本語で噛み砕きながら整理しています。

## 🎯 目標
- コードを「書き写す」のではなく「意味を理解して使う」ようになる。
- エイムズの街の生活背景から、住宅価格の謎を解き明かす。
- 過去の最高記録（RMSE: 0.13277）を、納得感を持って再現する。

## 🔍 捜査メモ（現在の進捗）
- [x] データの読み込み：まずは現場の台帳を開くところから。
- [ ] 基礎的な分析：家の広さと価格の関係をおさらい。
- [ ] 特徴量エンジニアリング：ユキパス流「雪かき」「ワケあり」視点の導入。

## 💡 難しい名前コード集
自分がつまずいたプログラミング用語を、自分なりの言葉で解説していきます。
- **RMSE**: 「予測と正解がどれくらいズレているか」の平均。
- **Feature Engineering**: 現場の知識を「新しい定規」にして測り直すこと。
